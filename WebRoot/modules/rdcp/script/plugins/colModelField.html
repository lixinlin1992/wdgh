格式化函数:
function(cell, options, row, tr, td) {
}
格式化函数的作用是在生成列表单元格时执行的方法，方法要求输出单元格最后显示的结果。
列表控件会提供(cell, options, row, tr, td)5个参数，其中:
cell:数据库查询返回的结果,如果不填写格式化函数默认将其作为单元格显示的结果
options:表格的列定义
row:该行的其他单元格查询结果数据,可以使用row["编码"]获取其他列的数据
tr,td:改行或该单元格的节点对象
